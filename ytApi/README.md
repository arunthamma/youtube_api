Prerequisites
The only prerequisite is Python. This API should be compatible with Python 3.x.

Installation

Install docker: https://docs.docker.com/install/linux/docker-ce/ubuntu/

Install docker-compose: https://docs.docker.com/compose/install/

Clone the repo to your home directory.
Run docker:
sudo docker-compose down; sudo docker-compose build; sudo docker-compose up

This will setup the backend web server as well as the postgres database.
